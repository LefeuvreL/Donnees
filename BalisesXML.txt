Relation d’Hyperonymie : 
Dans les balises, RG signifie Relation Générique

id_RelationGenerique : id du contexte dans la base de données
RG_Categorie : Catégorie à laquelle le marqueur testé appartient
RG_Marqueur : marqueur lexico-syntaxique testé (hors contexte)
RG_Contexte : contexte extrait avec le marqueur lexico-syntaxique (donc ce marqueur y apparait en théorie)
RG_Validation : Evaluation du candidat-marqueur : le candidat-marqueur, seul, permet-il d’indiquer la relation ? Oui/Non/Plutôt Oui/Plutôt Non
RG_Notes : quelques notes que j’avais prises concernant le contexte
RG_ReferenceSource : source du contexte, cad texte duquel il est issu
RG_Patron : le marqueur dans le contexte
RG_hypo : hyponyme
RG_hyperonyme : hyperonyme
